# LabRentPlatform


## 准备工作
安装依赖

```bash
pip install -r requirements.txt
```

迁移数据库

```bash
python manage.py makemigrations
python manage.py migrate
```

安装npm依赖

```bash
cd front_end
sudo npm link/ sudo npm install
cd ..
```
## 运行服务

启动Django服务（启动后端）

```bash
python manage.py runserver
```

启动vue-cli项目（启动前端）

```bash
cd front_end
npm install
sudo npm run serve
```

## 自动化测试

执行测试

```bash
python manage.py test core
```

统计测试覆盖率

```bash
coverage run manage.py test core
coverage html
```
