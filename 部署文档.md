# LabRentPlatform

安装依赖

```bash
pip install -r requirements.txt
```


迁移数据库
```bash
python manage.py makemigrations
python manage.py migrate
```

启动Django服务
```bash
python manage.py runserver
```


执行测试
```bash
python manage.py test core
```

统计测试覆盖率
```bash
coverage run manage.py test core
coverage html
```
